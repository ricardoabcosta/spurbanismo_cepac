Fluxo de deploy para Azure (até o Bloco 8 — AZURE_CREDENTIALS — ser configurado)

1. Faça o push normalmente — o CI roda e constrói as imagens no ACR (~3-4 min)

2. Verifique se o build terminou no GitHub Actions (ou aguarde o CI verde)

3. Execute o deploy:
   cd ~/Projetos/CEPAC
   ./scripts/deploy.sh

   Isso pega o SHA do commit atual, valida que a imagem existe no ACR,
   atualiza os 3 apps em paralelo e confirma a saúde ao final.

---

Para checar o estado a qualquer momento:
   ./scripts/deploy.sh --status

Mostra o que cada app está rodando vs o que existe no ACR, com OK ou DESATUALIZADO.

---

Resumo:

   git push origin main
            |
      CI roda (~4 min)
            |
   ./scripts/deploy.sh
            |
      Azure atualizado
